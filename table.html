HTML Tags, Definitions & Page Descriptions

This page lists definitions for all six website pages along with details of the HTML tags used across the portfolio.


Page Name / HTML Tag Purpose and Definition
<!DOCTYPE html> Tells the web browser that this document uses modern HTML5 formatting rules.
<html> Acts as the root container bounding all code elements on the webpage.
<head> Stores page metadata, configuration definitions, and the browser tab window title.
<title> Defines the explicit text header shown at the very top of the desktop browser tab view.
<body> Contains all the visible content, text, media, and layouts shown to the viewer.
<a> Anchor tag creating operational hyperlinks to other files using the href path pointer.
<h1>, <h3> Heading tags used to construct logical topic titles across varying layout sizes.
<p> Paragraph tag used to group plain text bodies together with vertical spacing.
<br> Inserts a simple line break to drop text down to the next row manually.
<hr> Generates a horizontal rule line across the screen to separate content sections.
<table> Creates a data layout grid composed of bounding rows and vertical data column intersections.
<tr> Table Row element used to structure single horizontally aligned item blocks.
<td> Table Data cell element that holds actual text values, images, or media parameters.
<th> Table Header cell element that applies centered bolding automatically for sorting titles.
<img> Embeds visual graphical picture documents using target resource paths via the src rule.
<audio> Loads sound files and provides real-time play, pause, and tracking control buttons to the user.
<video> Allocates an active display box to play media clips natively inside the workspace template.
<form> Wraps text entries and control buttons together to process user field submissions safely.
<input> Creates specific user input types based on assigned values (text, email, password, radio, checkbox, date, submit, reset).
<select> Spawns clean drop-down option lists to keep the structural layout compact.
